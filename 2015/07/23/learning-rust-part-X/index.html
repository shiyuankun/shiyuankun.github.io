rust学习笔记(十)

Universal Function Call Syntax

有些时候函数可能会有相同的名字。例如：

trait Foo {
    fn f(&self);
}

trait Bar {
    fn f(&self);
}

struct Baz;

impl Foo for Baz {
    fn f(&self) { println!("Baz’s impl of Foo"); }
}

impl Bar for Baz {
    fn f(&self) { println!("Baz’s impl of Bar"); }
}

let b = Baz;

这时候如果调用b.f()就会报错。必须这样

Foo::f(&b);
Bar::f(&b);

才能正常执行。rust在执行b.f()是会自动借用b作为&self，而这里不会，所以要手动将&b传入。

cates and modules

在rust里，crate就是library package的同义词。每个crate都有一个根module，根module下面还有很多子module。

在src/lib.rs中这么写定义模块

mod english {
    mod greetings {
    }

    mod farewells {
    }
}

mod japanese {
    mod greetings {
    }

    mod farewells {
    }
}

cargo会去src/.rs或者src//mod.rs中去找相应模块。

const and static

const N: i32=5;

定义了一个寿命为永久的N，并且在rust中const没有地址，因为所有的N调用在编译时都会被替换为具体值。

static N: i32=5;

同样定义了一个永久寿命的N，但这个本质上是一个变量。

static mut N: i32 = 5;

由于本身已经定义为static了，所以这样子可能导致一个线程读一个线程写，这是不安全的，所以要这么做只能放在unsafeblock里

unsafe {
    N += 1;

    println!("N: {}", N);
}

所有static的类型必然是sync的。

不同类型间转换

as可以做基本的转换

let x: i32 = 5;
let y = x as i64;

transmute可以不做检查直接转换

use std::mem;
unsafe {
    let a = [0u8, 0u8, 0u8, 0u8];
    let b = mem::transmute::<[u8; 4], u32>(a);
}

但还是会做一些基本的比如越界检查防止内存问题

联合类型

例如对于一个图，你可能有Node Edge两种类型，但这样不美观，所以就可以用trait把两种类型联合起来

trait Graph {
    type N;
    type E;

    fn has_edge(&self, &Self::N, &Self::N) -> bool;
    fn edges(&self, &Self::N) -> Vec<Self::E>;
    // etc
}

并实现相关的函数

struct Node;

struct Edge;

struct MyGraph;

impl Graph for MyGraph {
    type N = Node;
    type E = Edge;

    fn has_edge(&self, n1: &Node, n2: &Node) -> bool {
        true
    }

    fn edges(&self, n: &Node) -> Vec<Edge> {
        Vec::new()
    }
}

不定长类型

大多数类型都有固定长度，但rust也允许使用动态长度的类型。但这里有一定的限制

1.只能通过指针操作unsized type, &[T]可以但[T]不行
2.变量和参数不能有动态长度的类型
3.只有struct的最后一栏允许动态长度类型。enum不允许有动态长度类型

操作符重载

只要对一个类型实现相应的函数就能实现相应重载。

use std::ops::Add;

#[derive(Debug)]
struct Point {
    x: i32,
    y: i32,
}

impl Add for Point {
    type Output = Point;

    fn add(self, other: Point) -> Point {
        Point { x: self.x + other.x, y: self.y + other.y }
    }
}

fn main() {
    let p1 = Point { x: 1, y: 0 };
    let p2 = Point { x: 2, y: 3 };

    let p3 = p1 + p2;

    println!("{:?}", p3);
}

这些允许重载的函数可以在std::ops中找到
